## 🔍 Comandos investigados

### git log --oneline
Muestra historial de los commits, mostrando ID de cada commit.

### git remote -v
Muestra los repositorios remotos configurados para este proyecto, indicando URL para Descargar y Subir.

### git branch
Muestra las ramas del repositorio. La rama en la que estás actualmente con un asterisco (*) . También sirve para cambiar de rama o para crearlas.

### git pull
Descarga los cambios que se han hecho en GitHub y los combina con los de tu repositorio local. Es útil para mantener tu copia actualizada cuando se trabaja en grupo. 

### .gitignore
Archivo que indica a git qué archivos o carpetas **ignorar** .

### git add . vs git add archivo
Git add . Añade todos los archivos modificados desde la carpeta actual (cd) y Git add archivo añade sólo ese archivo al próximo commit.
